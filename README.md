# 🎵 Music Backend App

## 🚀 Features
- User Authentication (JWT)
- Role-based access (User / Artist)
- Music Upload (ImageKit)
- Album Creation
- Secure APIs

## 🛠 Tech Stack
- Node.js
- Express.js
- MongoDB
- Mongoose
- ImageKit
- JWT Auth
